# data-structure-and-algorithm
## 常见时间复杂度为O(n)的算法
- 双指针算法
- 打擂台算法（求最大/小值）
- 单调栈算法
- 单调队列算法

## 常见时间复杂度和对应的算法
O(1) < O(log n) < O(n) < O(nlog n) < O(n^2) < O(n^3) < O(2^n) < O(n!)
- O(log n) 二分法比较多
- O(n^1/2) 分解质因数（极少）
- O(n) 双指针、单调栈、枚举法
- O(nlog n) 排序
- O(n^2)｜O(n^3) 动态规划等
- O(2^n) 组合类的搜索问题
- O(n!) 排列类的搜索问题
